#### **install**

1. api-gateway에 설치
   yarn add @apollo/gateway
   yarn add @nestjs/apollo
   yarn add @nestjs/graphql
   yarn add apollo-server-express
   yarn add graphql

2. services에 설치
   yarn add @apollo/federation
   yarn add @apollo/subgraph
   yarn add @nestjs/apollo
   yarn add @nestjs/graphql
   yarn add apollo-server-express
   yarn add graphql

3. 모든 main.ts 설정 => listen port
4. services resolver 생성
5. module 설정
6. 도커 서버 실행 테스트
